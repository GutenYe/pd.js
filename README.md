Overview
--------

For browser

	# load pd.js
	pd 1, 2, 3


For node

	require 'pd'
	pd 1, 2, 3


Internal is using `console.log`.

Easy for search with `$ grep 'pd ' -R lib`.


Install
----------

For bpm

	$ bpm add pd

for Node.js

	$ npm-g install pd
